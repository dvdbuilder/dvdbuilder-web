DVD Authoring

  • Define DVD structure with XML
  • Assign first play command
  • Multiple titles (up to 99)
  • Multiple video objects (VOB) per title (up to 10)
  • Multiple chapters per title
  • End play commands (jump to title, jump to menu)
  • DVD Virtual Machine Commands

DVD Menus

  • Create disc menu (VMGM)
  • Create video titleset root menus (VTS)
  • Create title menus
  • Use MPEG still picture or movie for menu background
  • Define up to 36 buttons per menu
  • Define buttons mask, transparency and highlight color
  • Navigate between titles from different title sets

Output

  • NTSC and PAL TV formats supported
  • 4:3 and 16:9 aspect ratio supported
  • Built-in MPEG-2 multiplexer
  • Built-in subtitle encoder
  • Outputs a finished VIDEO_TS directory with VOB, IFO and BUP files
  • Use PrimoBurner™ for DVD burning (requires a separate PrimoBurner license)

Video Input

  • MPEG-2 program stream
  • MPEG-2 elementary video stream
  • DVD Video OBject file (VOB) import
  • Read from a callback interface or directly from a file
  • Includes AVBlocks™ - a production quality MPEG-2 encoder

Audio Input

  • MPEG-2 elementary audio
  • AC3 (Dolby Digital) elementary audio
  • DTS elementary audio
  • Detect multiple audio streams and assign them to specific language
  • Read from a callback interface or directly from a file

Subtitles

  • Subpicture stream (encoded as MPEG-2 PS)
  • Subtitles in Spruce Technologies STL format
  • Detect multiple subtitle streams and assign them to specific language
  • Read from a callback interface or directly from a file
